EPiC — 졸업요건 자동 진단 · AI 커리큘럼 추천

경희대 소프트웨어융합대학 3개 학과(CS·AI·SWCON) 2019~2025학번 교과목 데이터를 AI가 학습, PDF 성적표 업로드만으로 졸업요건을 자동 진단하고 키워드 기반 맞춤 수강 과목을 추천하는 서비스입니다.

핵심 기여 1 — GPT Vision 자동화 파이프라인
  • 정규식 파서 → GPT Vision API 기반으로 전환: 레이아웃 무관 교과목 데이터 추출
  • RestTemplate → WebClient 전환: 비동기 HTTP 호출로 AI 서버 위임 구조 개선
  • CustomException으로 Spring Boot / FastAPI 에러 응답 형식 통일
핵심 기여 2 — Redis 2-Phase 캐싱
  • Phase 1: PDF 콘텐츠 MD5 해시 기반 졸업진단 결과 캐싱 (24h TTL)
  • Phase 2: 사용자 세션별 커리큘럼 추천 캐싱 (6h TTL) + 30분 세션 TTL 분리
  • 커리큘럼 추천 응답시간 5.8s → 0.07s (99% 단축) · Gzip 응답 페이로드 40% 감소
DevOps
  • GitHub Actions CI/CD 병렬 빌드 · 레이어 캐싱 · Microsoft Azure 기반 배포
  • 신입 2명 포함 6인 팀 온보딩 주도
