민들레 — 동대문구 통합 민원 자동화 플랫폼

동대문구 이문뉴타운(2027년 1.3만 세대 예정) 주민의 민원을 AI가 적절한 채널 (안전신문고·시장게시판·전자청원)으로 자동 분류·초안 생성하는 서비스입니다.

핵심 기여 — 이기종 4서버 오케스트레이션
  • Spring Boot(8443) · FastAPI(8000) · Node.js(3000) 묶는 오케스트레이션 API 직접 설계
  • 상태 플래그 기반 이벤트 구조로 DB 정합성 100% 확보 (Request Chaining 최적화)
  • LangChain + GPT SSE 스트리밍으로 민원 초안 실시간 생성
  • Node.js Data Bridge: 새울 행정 시스템·동대문구 포털 스크래핑 → 민원 데이터 수집
아키텍처
